Reeve. A Simple, Straightforward Language made to handle Data, File writing, and more.

ReeveScript is a lightweight library that contains features like

  • Data Handling
  • File Checking
  • Formatting
    • < And a Lot more.

    BACKSTORY: Reeve is a simple Framework made in c++ used for generally writing scripts for mostly anything. And while reeve is its own language, i'd say it'd depend on about 50-60% of the C API to function how it should.

    It was made to basically turn c++ Into easier Syntax, And sort of make a "comfortable" Modern type Syntax which you may see in languages like Python, Ruby, Etc. . .

    If you're reading this, im guessing you've probably already set things up and you're ready to code.

    There are many ways to go about this, one way is to just open up the .sln file inside of it, create a new file in the "ext" Folder, Then Include the reeve header/piler file, then you have every function in your possession.

    Like i said, there are many ways to do the Reeve Things in C++ and C, But i felt C/C++ Syntax Would've been easier to read if i just bundled it up and made it with certain functionalities.
